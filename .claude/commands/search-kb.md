`$ARGUMENTS` 키워드로 `docs/` 하위의 모든 마크다운 파일을 검색해주세요.

## 검색 방법

1. 파일명에서 키워드 매칭
2. 파일 내용에서 키워드 매칭 (섹션 제목, 개요 우선)
3. 관련도 순으로 정렬

## 출력 형식

```
## 검색 결과: "{키워드}"

### 직접 매칭 (파일명 포함)
- `docs/{category}/{file}.md` — {한 줄 설명}

### 내용 매칭 (본문 포함)
- `docs/{category}/{file}.md` (섹션: {섹션명}) — {관련 내용 요약}

### 관련 문서 (간접 연관)
- `docs/{category}/{file}.md` — {연관 이유}
```

검색 결과가 없으면 유사한 키워드 또는 관련 카테고리를 제안해주세요.
