# cilium-practice 문서

이 디렉토리는 주제별 번호 체계로 문서를 정리합니다. 앞 번호는 학습과 운영 흐름을 나타내며, 90번대는 저장소 관리용 문서입니다.

## 문서 구조

- [01-installation](./01-installation/): Cilium 설치와 업그레이드 - Cilium 설치, 업그레이드, 실행 환경 준비 문서를 모읍니다.
- [02-architecture](./02-architecture/): Cilium 아키텍처 - Cilium 구성 요소와 동작 구조 문서를 모읍니다.
- [03-networking](./03-networking/): 네트워킹 - BGP, ENI, Egress Gateway, Load Balancing 문서를 모읍니다.
- [04-security](./04-security/): 보안 - NetworkPolicy, FQDN 정책, WireGuard 문서를 모읍니다.
- [05-observability](./05-observability/): 관측성과 모니터링 - Hubble, Prometheus, Grafana 문서를 모읍니다.
- [06-service-mesh](./06-service-mesh/): 서비스 메시 - Cilium 서비스 메시 문서를 모읍니다.
- [07-operations](./07-operations/): 운영과 실습 - 디버깅, 문제 해결, E2E 실습 문서를 모읍니다.
